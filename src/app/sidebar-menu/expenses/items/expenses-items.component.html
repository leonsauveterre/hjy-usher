本文会记录什么
本文主要包括 bill_enum 表中的特殊费用项（其实是功能配置项），用户无法访问，而且与子表（bill_enum_sub）无关。其他内容如有需要，后期可以随时添加。
特殊费用项
是否显示刷新按钮
enum_name = 'is_show_refresh_button' 用于控制页面是否显示刷新按钮，利用 enum_message （true 或 false）字段控制具体值。
是否开放时间限制
enum_name = 'is_open_time' 用于控制是否开放导入、回刷等操作的时间限制，利用 enum_message （true 或 false）和 remark （英文逗号分隔）字段共同控制具体值。
标记标准单数量限制
enum_name = 'mark_standard_limit' 用于控制标准单最大数量（上限），利用 enum_code 字段控制具体值。
标记账号数量限制
enum_name = 'mark_account_limit' 用于控制账号最大数量（上限），利用 enum_code 字段控制具体值。
标记标准单总数量限制
enum_name = 'mark_total_standard_limit' 用于控制标准单总数量最大值（上限），利用 enum_code 字段控制具体值。
标记账号总数量限制
enum_name = 'mark_total_account_limit' 用于控制账号总数量最大值（上限），利用 enum_code 字段控制具体值。
将某些存在于金额字段的不可解析值归零
enum_name = 'tolerable_abnormal_decimal' 用于将导入账单时遇到的特殊金额字符串转为零，利用 remark （英文逗号分隔）字段控制具体值。
某些租户导入自定义营销账单时不进行已有时间校验
enum_name = 'custom_marketing_import_trustworthy' 用于取消某些租户导入自定义营销账单时的时间限制，利用 remark （英文逗号分隔）字段控制具体值。
为某些租户私有化公共规则
enum_name = 'privatize_rules_for' 用于为特定的租户私有化 version_no = 86 标记的公共规则，利用 remark （英文逗号分隔）字段控制具体值。
