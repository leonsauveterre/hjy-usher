1. The Outer Container
Every blog must have a unique identity. Wrap it with an outer container using a unique id and class blog-container mx-auto. This keeps the layout centered and consistent. Example:
2. Titles and Headlines
The Main Title
Use a div with the class blog-title for your blog title. Ensure it's captivating and reflects the content. Example:
Subtitles
Use blog-sub-title for subtitles. While optional, adding the author's name and last updated time is highly recommended for personalization. Example:
3. Paragraph Structuring
Wrapper
Each paragraph is contained within a div using the class blog-paragraph. This ensures clear separation between ideas. Example:
Headers
Headers should be composed by blog-header-1, blog-header-2, or blog-header-3. However, for better comprehensibility, I wouldn't use blog-header-3 personally because that would be too deep a level for a simple blog. That said, you can always use Delimiters & Tags (see below). Example:
Texts
Regular text should go in blog-text-normal. Example:
Delimiters & Tags
Use blog-highlight-canary for delimiters, often accompanied by blog-highlight-compact tags for a quirky feel. Example:
Tables
Use Bootstrap's table-striped and table-bordered classes for tables. Headers, rows, and cells don't require custom styles. Example:
Highlighter
For highlights, use blog-highlight-canary or blog-highlight-navy. Preferably, I would use blog-highlight-navy more because the former one is initially designed for Delimiters & Tags, but it's up to you anyway. Example:
Q & A Boxes
Use blog-question-title for the question and alert alert-success blog-question-body for the answer. Example:
